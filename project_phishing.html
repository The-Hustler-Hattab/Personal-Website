Phishing Project

Informational site about phishing

This project utilizes Angular for the frontend and a serverless api through AWS SAM for the backend.

Key Components:

  • Frontend: Angular
  • Backend: AWS SAM API

Project Overview:

The main goal of this project is to educate individuals about the risks associated with phishing attacks. It includes:

  • Real-world examples of phishing pages, including mimicked pages of popular services like Google and Facebook to showcase the potential dangers.
  • Tips and best practices for avoiding falling victim to phishing scams.

Technology Stack:

  • Frontend:
    • Angular Framework
    • Bootstrap for UI
    • GitHub Pages for Hosting
  • Backend:
    • DynamoDB Database
    • API Gateway
    • Python Lambda Function
