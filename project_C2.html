Command & Control

Control a fleet of computers with a simple UI

This project utilizes Angular for the frontend, Java Spring for the backend, and java for the agent. Authentication is handled through Okta, and communication between the frontend and backend is secured using JWT tokens issued by Okta.

Key Components:

  • Frontend: Angular
  • Backend: Java Spring
  • Authentication: Okta
  • File Storage: S3
  • Database: MySQL

Project Overview:

The Command and Control Project is designed to manage pawned computers using command line emulator in frontend with the abilty to harvest credientials, access camera, and exfiltrate files.

Technology Stack:

  • Frontend:
    • Angular Framework
    • Bootstrap for UI
    • Amazon S3, and Cloud Front for Hosting
  • Backend:
    • Java Spring Framework
    • AWS S3
    • MySQL Database
    • Elastic Beanstalk for Hosting
  • Agent:
    • Java Maven Reverse Shell
    • C# .NET Reverse Shell
  • Authentication:
    • Okta Authentication Service
